# Security Policy

## Supported Versions

The latest `main` branch and tagged releases are supported.

## Reporting A Vulnerability

Please use GitHub Security Advisories or open a minimal public issue that does not include secrets, private paths, raw sessions, customer data, account data, tokens, database URLs, webhook URLs, cookies, passwords, or private keys.

## Project Boundary

This starter kit is local-first and single-user by default. It does not provide hosted authentication, multi-tenant isolation, or remote ingestion controls.

Never connect automatic ingestion to a private workspace without an explicit allowlist and a safety scan.

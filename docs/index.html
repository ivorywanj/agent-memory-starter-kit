JourneyMem Start

Start JourneyMem without making your Agent inspect a repo.

Start with the JourneyMem menu. Open Codex, TRAE Work, Claude Code, Cursor, or another Agent, then type memory.

What do you want to do?
1. memory new - Create a memory library
2. memory connect - Connect this Agent to an existing memory library

Other command:
- memory backup - Back up a memory library
Copy Agent prompt Command not available? Back up memory

After the menu

Use memory new Create your first memory library only after choosing the new-library path.
Use JourneyMem in my current Agent Connect this Agent to an existing memory library without copying memory files.
Back up or move my memory library Create a zip backup, then connect from the new computer.

Command not available?

If the memory command is not available yet, run this once on your computer. It installs the memory command and Agent helper files.

curl -fsSL https://ivorywanj.github.io/agent-memory-starter-kit/install.sh | bash

After install, open your Agent and type:

memory

Copy an Agent-ready prompt

Click a copy button when you need to start from an Agent message. The Agent should install or activate JourneyMem first, then show the menu.

Backup

Use this when moving to another computer. Backup creates a zip file; it does not connect or switch Agents.

memory backup

You can use the default backup folder or choose a custom save path.
